This is assorment of various tools I've written in JavaScript. Some are useful, others are not, most should work, but some rely on external services that might have died since I last played with them.

All of these tools are hosted on Github. They are Open Sourced and Free Software available on my [Github account](https://github.com/totu/).

<div style='width:100%;border-top:dashed 1.2px rgb(219,219,219,0.9);height:0'></div>
# [Morse](/morse)

Utility to practice morse alphabet.

<div style='width:100%;border-top:dashed 1.2px rgb(219,219,219,0.9);height:0'></div>
# [Workout](/workout)

Workout tracker for gym. This is really outdated, but someone might be inspired by it, so I've left it up.

<div style='width:100%;border-top:dashed 1.2px rgb(219,219,219,0.9);height:0'></div>
# [Transformer](/transformer)

Easy string manipulation. This tools helps you solve trivial [CTF](https://en.wikipedia.org/wiki/Capture_the_flag#Computer_security) challenges.

<div style='width:100%;border-top:dashed 1.2px rgb(219,219,219,0.9);height:0'></div>
# [IP mangler](/ip/)

This is related to my [notes on IP addresses](/notes-on-ip-addresses). A tool to mangle IP addresses in weird ways to get around IP based blocks.

<div style='width:100%;border-top:dashed 1.2px rgb(219,219,219,0.9);height:0'></div>
# [Chart](/chart)

Simple plotter. Often I find myself wanting to plot out some data, but without a simple way of displaying it. This tool is very simple and fast chat plotter that fills my needs most of the time.

<div style='width:100%;border-top:dashed 1.2px rgb(219,219,219,0.9);height:0'></div>
# [Voice](/voice)

VoIP without accounts and servers. This relies on [PeerJS](https://peerjs.com/) and their public server uptime is spotty at best.

<div style='width:100%;border-top:dashed 1.2px rgb(219,219,219,0.9);height:0'></div>
# [EXIF dumper](/exif)

Simple EXIF data dumper.

<div style='width:100%;border-top:dashed 1.2px rgb(219,219,219,0.9);height:0'></div>
# [Hexdump](/hexdump)

Simple hex dumper.

<div style='width:100%;border-top:dashed 1.2px rgb(219,219,219,0.9);height:0'></div>
# [Portscan](/portscan)

This was proof-of-concept demo for browsers ability to scan for open ports. The time delay between closed/blocked and open ports was significat. However this has been fixed in all major browsers, but I've left it up for fun.
